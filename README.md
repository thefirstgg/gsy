# 在线代码判题系统
项目介绍：基于Spring Boot + Spring Cloud 微服务 + Docker 的在线代码判题系统。  
1.管理员可以创建，管理题目；用户可以阅读题目，编写题目，提交代码，查看提交结果。  
2.系统根据设定好的题目测试用例在代码沙箱中对代码编译，运行，判断输出是否正确。代码沙箱接口可以给其他用户使用。
